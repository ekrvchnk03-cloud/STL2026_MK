# Мастер-класс «Интернет-гигиена»

Веб-приложение для проведения мастер-класса на тему цифровой безопасности.

## 🚀 Запуск

### Локально
Открыть `index.html` в браузере. Никаких зависимостей не требуется.

### GitHub Pages
1. Создать репозиторий на GitHub
2. Залить файлы в ветку `main`
3. В настройках репозитория → Pages → выбрать ветку `main`
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo-name>/`

## 📁 Структура

- `index.html` — лендинг с регистрацией (ловушка)
- `dashboard.html` — личный кабинет
- `poll.html` — опрос
- `block-1.html` — раздел 1 (соглашение)
- `block-2.html` — раздел 2 (cookies)
- `block-3.html` — раздел 3 (разрешения)
- `checklist.html` — финальный чек-лист
- `admin.html` — скрытая админ-панель для спикера

## ⚠️ Концепция

Сайт — **пустышка**. Данные пользователей **не сохраняются на сервере**.
Всё хранится только в `localStorage` браузера для UX.

В конце мастер-класса спикер показывает эту особенность как часть демонстрации.

## 🛠 Стек

- HTML + Tailwind CSS (via CDN)
- Vanilla JavaScript
- localStorage для прогресса
